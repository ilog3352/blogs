放异步

一帧或者是一个tick会做六件事

  1. 处理用户的事件，event 例如click,input,chang
  2. 执行event loop
  3. 执行 requestAnimationFrame （动画）
  4. 执行dom 回流与重绘
  5. 计算更新图层的绘制指令
  6. 绘制指令合并主线程，有空余时间会加载分包的代码
